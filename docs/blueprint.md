# **App Name**: Visionary Care

## Core Features:

- Daily Eye Health Log: Record daily symptoms with a 0-10 rating scale and timestamps, viewable in a historical log.
- AI Eye Health Tool: Receive personalized eye care advice based on symptom logs, including suggestions for screen time, blinking, and consultations. Initial implementation will be rule-based with a tool for LLM integration.
- Guided Eye Exercises: Access a list of guided exercises like the 20-20-20 rule, eye rotation, and palming, each with a short description.
- Food Reminder: Daily reminders and tips about foods rich in Vitamin A, Omega-3, Lutein, and Zeaxanthin.
- Spectacle Support: Tailored advice for glasses wearers, including reminders for lens power checks, cleaning tips, and screen glare reduction strategies.

## Style Guidelines:

- Primary color: Soft lavender (#D0BFFF) to promote a sense of calm and well-being.
- Background color: Light gray (#F0F0F4), desaturated hue for a clean interface.
- Accent color: Pale blue (#ADD8E6) for interactive elements.
- Body and headline font: 'Inter' sans-serif for a modern and objective feel.
- Simple, outline-style icons related to eye health and wellness.
- Subtle transitions and animations to guide users through the app.
- Mobile-first, component-based design.
